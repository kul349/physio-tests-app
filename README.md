**PhysioTests** is a professional-grade web application designed to bridge the gap between clinical physiotherapy assessments and patient understanding. It provides a standardized library of orthopedic and neurological tests, empowering practitioners with accurate protocols and patients with clear education.

## üéØ Purpose
Physical therapy can often feel overwhelming for patients. PhysioTests simplifies this by providing:
* **Standardized Protocols:** Ensuring clinicians follow evidence-based steps.
* **Visual Learning:** Integrated video demonstrations and diagrams.
* **Empowerment:** Reducing patient anxiety through "What to Expect" guides.

## üõ†Ô∏è Key Features

### 1. Clinical Test Library
Categorized assessments for high-speed navigation in a clinical setting:
* **Lower Body:** ACL/PCL stability, Meniscus tests, Gait analysis.
* **Upper Limb:** Rotator cuff integrity, Grip strength, Elbow stability.
* **Spine & Posture:** Flexibility profiling and postural alignment.
* **Neurological:** Reflex evaluations and dermatome/myotome mapping.

### 2. Patient Education Hub
* **Assessment Stages:** A step-by-step breakdown of the evaluation process.
* **Resource Downloads:** Access to the "Patient‚Äôs Guide to Physical Therapy" (PDF).
* **Recovery Tracking:** Tools to help patients visualize their progress.

### 3. Professional Tools
* **Manual v2.4:** A comprehensive digital handbook for advanced practitioners.
* **Mobile-First Design:** Fully responsive UI for use on tablets and phones during active therapy sessions.

## ‚öôÔ∏è Development Setup

To run this project on your local machine:

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/kul349/physio-tests-app.git](https://github.com/kul349/physio-tests-app.git)
    ```

2.  **Install Dependencies**
    ```bash
    cd physio-tests-app
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open the local URL provided in your terminal (usually `http://localhost:5173`) to see the result.

4.  **Build for Production**
    ```bash
    npm run build
    ```
